# 3D打印耗材进销存管理系统

一个功能完整的3D打印耗材库存管理解决方案，支持耗材管理、供应商管理、打印费用预估和花费统计。

## 功能特性

### 1. 耗材管理
- 完整的CRUD操作（创建、读取、更新、删除）
- 库存状态自动追踪（充足、一般、不足、已用完）
- 支持按材料类型、供应商筛选
- **一键复制功能**：点击复制按钮快速创建相同规格的新耗材
- 颜色可视化展示

### 2. 供应商管理
- 供应商信息维护（名称、联系人、电话、邮箱、地址）
- 耗材与供应商关联
- 采购费用统计

### 3. 打印费用预估
- 自定义价格计算公式
- 材料成本按类型区分（PLA、PETG、ABS、TPU、ASA）
- 包含：材料费、人工费、电费、基础费用
- 可调节利润率
- 历史预估记录

### 4. 花费统计
- 费用分类记录（耗材购买、设备配件、电费、维护、其他）
- 月度趋势图表
- 按类型分布统计
- 筛选和搜索功能

### 5. 数据持久化
- JSON数据库存储
- 刷新页面不丢失数据
- 自动备份机制

## 技术栈

- **后端**: Node.js + Express
- **前端**: HTML5 + Tailwind CSS + JavaScript
- **数据库**: JSON文件存储
- **图标**: Font Awesome

## 从 GitHub 获取并运行

```bash
# 1. 克隆仓库
git clone https://github.com/robber89757/Filament-Management.git

# 2. 进入项目目录
cd Filament-Management

# 3. 安装依赖（需要先安装 Node.js，下载地址：https://nodejs.org）
npm install

# 4. 启动服务器
node app.js

# 5. 打开浏览器访问
# http://localhost:3001
```

> **前提条件**：需要安装 [Node.js](https://nodejs.org)（推荐 LTS 版本）

## 快速开始

### 方式一：Windows 一键启动（推荐）

1. 双击运行 `启动.bat` 文件
2. 等待窗口显示"服务器已启动"
3. 打开浏览器访问: http://localhost:3001

### 方式二：命令行启动

1. 打开命令行，进入项目文件夹
2. 运行以下命令：

```bash
# 安装依赖（只需运行一次）
npm install

# 启动服务器
node app.js
```

3. 看到"服务器已启动，运行在端口：3001"后
4. 打开浏览器访问: http://localhost:3001

### 方式三：双击启动（创建快捷方式）

1. 右键点击 `app.js` 文件
2. 选择"创建快捷方式"
3. 把快捷方式放到桌面
4. 双击快捷方式即可启动

## 页面说明

| 页面 | 文件 | 功能 |
|------|------|------|
| 主页 | index.html | 统计概览、快速操作入口 |
| 耗材管理 | filament.html | 耗材列表、新增、编辑、复制、删除 |
| 供应商 | supplier.html | 供应商列表和信息管理 |
| 费用预估 | calculator.html | 打印成本计算和公式设置 |
| 花费统计 | expense.html | 费用记录和统计图表 |
| 销售情况 | sales.html | 销售记录和利润统计 |

## 数据库文件

数据存储在 `data/spool.db` 文件中，包含以下集合：
- spools: 耗材信息
- suppliers: 供应商信息
- expenseRecords: 费用记录
- usageRecords: 使用记录
- priceFormulas: 价格公式配置

## 使用提示

1. **添加耗材**: 进入耗材页面，点击"添加耗材"按钮，填写信息后保存
2. **复制耗材**: 在耗材列表中点击复制图标，自动填充表单创建新耗材
3. **设置价格公式**: 进入费用预估页面，点击右上角设置按钮调整各项费率
4. **记录花费**: 进入花费统计页面，点击"添加记录"记录采购支出
5. **管理供应商**: 进入供应商页面，添加和管理耗材供应商信息

## 注意事项

- 确保端口3001未被占用
- 定期备份 `data/spool.db` 文件
- 删除供应商不会删除关联的耗材，只会解除关联
